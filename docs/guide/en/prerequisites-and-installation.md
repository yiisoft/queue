# Prerequisites and installation

## Requirements

- PHP 8.1 or higher.
- PCNTL extension for signal handling (optional, recommended for production use).

## Installation

Install the package with [Composer](https://getcomposer.org):

```shell
composer require yiisoft/queue
```
